# Android Timer

**Android Timer** is a lap‑based running timer for **Android** designed to make tracking your runs simple. 
Features include:

* **Physical button controls** using the device’s volume keys: press _Volume Down_ to start a lap or resume a run, and _Volume Up_ to stop or pause.
* **Real‑time metrics** including total run duration, current lap time, lap count, and distance covered.
* **Detailed lap breakdowns** showing each lap’s duration and speed.
* **Flexible lap management** with the option to delete any lap and automatically merge its time into the next one.
* **Run history** that lets you review past sessions and resume any saved run.
* **Configurable distance settings** to define how many laps make up a mile, kilometer, or any custom distance unit.
* **Live and saved run statistics** for quick insight into performance during a run.
